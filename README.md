# Workday Insight

เว็บแอป Google Apps Script สำหรับสรุปเวลาสาย คะแนนเวลาสาย และวันลา โดยเก็บข้อมูลใน Google Sheets

## ความสามารถ

- เลือกช่วงวันที่และกรองตามพนักงาน
- เพิ่ม แก้ไข และลบพนักงาน ยอดรวมเวลาสายรายเดือน และวันลา
- คำนวณคะแนนเวลาสายแยกในแต่ละเดือนโดยอัตโนมัติ
- เมื่อเลือกหลายเดือน ระบบแสดงคะแนนเฉลี่ยรายเดือนของพนักงานและภาพรวม
- คำนวณสิทธิ์ลาคงเหลือและส่งออกสรุปเป็น CSV

## เกณฑ์คะแนนเวลาสายต่อเดือน

| เวลาสายรวม | คะแนน |
|---|---:|
| 0–29 นาที | 100 |
| 30–59 นาที | 90 |
| 60–89 นาที | 80 |
| 90–119 นาที | 70 |
| 120 นาทีขึ้นไป | 60 |

คะแนนคำนวณขณะอ่านข้อมูล ไม่ต้องเพิ่มคอลัมน์คะแนนใน Google Sheets

## ติดตั้งใน Google Sheets

1. เปิด Google Sheet ที่ใช้เป็นฐานข้อมูล แล้วเลือก **Extensions → Apps Script**
2. อัปเดตไฟล์ `Code.gs`, `index.html`, `Styles.html`, `Scripts.html` และ `appsscript.json` จากโปรเจกต์นี้
3. ตั้งค่า Time zone เป็น `Asia/Bangkok`
4. รันฟังก์ชัน `setupDatabase` หนึ่งครั้งและอนุญาตสิทธิ์ ระบบจะสร้างชีต `employees`, `attendance` และ `leave`
5. เลือก **Deploy → New deployment → Web app** แล้วตั้งค่า Execute as: Me และสิทธิ์การเข้าถึงให้เหมาะกับองค์กร
6. เมื่อแก้โค้ด ให้สร้าง deployment version ใหม่เพื่อให้ผู้ใช้เห็นเวอร์ชันล่าสุด

เวลาสายบันทึกเป็นยอดรวมของพนักงานต่อเดือน โดยใช้วันที่ 1 ของเดือนในชีต `attendance` หากบันทึกพนักงานและเดือนเดิม ระบบจะอัปเดตยอดเดิม

## สิทธิ์ลาเริ่มต้น

- ลาป่วย 30 วันต่อปี
- ลากิจ 5 วันต่อปี
- พักร้อนตามอายุงาน: ต่ำกว่า 1 ปี = 0 วัน, 1–2 ปี = 6 วัน, 3–5 ปี = 8 วัน, 6–8 ปี = 10 วัน, 9–11 ปี = 12 วัน, 12–14 ปี = 14 วัน และ 15 ปีขึ้นไป = 15 วัน
